
# computing_network

한양대학교 erica 컴퓨터 네트워크 수업에서 자료입니다.
처음으로 만드는 것이여서 인터넷을 참고하면서 만들었습니다



1. git clone을 하거나 코드나 복사
2. 실행
  
a. gcc. sockat.c를 입력
    => 입력을 하면 sockat이 실행이 된다. 실행 된 것을 확인하는 법
  
b. ./a.out port number
    => 위와같이 port number 대신에 수를 쓰면 포트번호에서 실행을 시킬 수 있다 
      저는 9000번에서 9009번 포트 사이에서 실행 하였습니다.
